## 主要使用了
  * axios、jsonp获取数据
  * babel 系列使用es6语法编写
  * better-scroll 由一款滚动插件
  * classnames: react 全家桶一分子，做class的绑定工作
  * eslint 做语法规范检查
  * scss: 做css 预编译
  * redux: 本地数据管理
  * react-router-dom: 路由管理
  * react-transition-group: 页面切换效果
  * react-lazyload: 图片懒加载
